Documentation

RabbitJS

A modern DOM manipulation library that surpasses jQuery — with reactive data binding, built-in observers, a template engine, animation timeline, and full jQuery plugin compatibility.

v1.0.0 jQuery 3.7.1 Compatible Zero Dependencies MIT License ES2020+

Installation

Script Tag
<!-- Drop-in. Exposes window.RabbitJS, window.jQuery, and window.$ -->
<script src="rabbit.js"></script>
CommonJS / Node
const $ = require('./rabbit.js');
AMD / RequireJS
define(['rabbit'], function($) { /* ... */ });

After loading, window.$, window.jQuery, and window.RabbitJS all point to the same function. Existing jQuery plugins will work without modification.

Quick Start

// Standard jQuery syntax — works unchanged $(function() { $('.btn').on('click', function() { $(this).toggleClass('active').text('Clicked!'); }); $.getJSON('/api/data').then(data => { $('#output').html(`<p>${data.message}</p>`); }); }); // RabbitJS-exclusive reactive binding $('#counter').bind('count', function(val, $el) { $el.text(`Count: ${val}`); }).setReactive('count', 0);

jQuery Compatibility

JQ
RabbitJS exposes window.jQuery and window.$ and reports $.fn.jquery = "3.7.1". Any jQuery plugin that checks the version string, uses $.fn.extend(), $.Callbacks, $.Deferred, $.event, $.cssHooks, or $.fx will work without modification.
// jQuery plugins register normally $.fn.myPlugin = function(options) { return this.each(function() { $(this).css('color', options.color || 'red'); }); }; // Use it immediately $('.text').myPlugin({ color: 'blue' }); // noConflict works too const R = $.noConflict(true); // restores both $ and jQuery globals

Selection & Initialization

CORE

The main $() / RabbitJS() function accepts CSS selectors, HTML strings, DOM elements, NodeLists, arrays, or a ready callback.

$(selector) $(selector [, context])
CORE

Creates a RabbitCollection. Accepts a CSS selector string, HTML string, DOM element, NodeList, array of elements, or a ready callback function.

$('.items') // CSS selector $('<div class="box">') // HTML creation $(document) // document/window wrapping $([el1, el2]) // array of elements $('li', '#list') // scoped to context $(function() { ... }) // DOM ready callback
→ returns RabbitCollection
.ready(fn)
CORE

Executes fn when the DOM is ready. If already ready, fires immediately.

$(document).ready(function() { /* DOM ready */ });

Traversal

CORE
.find(selector)

Finds all descendants matching selector within the collection.

$('#nav').find('a.active')
.closest(selector)

Travels up the DOM, returning the nearest ancestor matching selector (including the element itself).

$('.btn').closest('.card')
.parent([selector]) / .parents([selector])

.parent() returns immediate parents. .parents() returns all ancestors up to <html>. Optionally filtered by selector.

.children([selector])

Returns direct children, optionally filtered.

$('ul').children('li.active')
.siblings([selector])

Returns all siblings of each element, excluding the element itself.

.next([sel]) / .prev([sel]) / .nextAll([sel]) / .prevAll([sel])

Navigate siblings. next/prev get the adjacent sibling; nextAll/prevAll get all following/preceding siblings.

.filter(selector|fn) / .not(selector|fn)

Reduce the set to matching elements. .not() is the inverse.

.is(selector)

Returns true if any element in the collection matches. Also accepts a function or DOM element.

.has(selector)

Filters to elements that contain a descendant matching selector.

.eq(n) / .first() / .last()

Returns the element at index n (negative counts from end), or the first/last element, as a new collection.

.get([n]) / .toArray()

.get(n) returns the raw DOM element at index. .get() / .toArray() return a plain array of all elements.

.index([target])

With no argument, returns the index of the first element among its siblings. With a target, returns the position of that target in the collection.

.each(fn)

Iterates over each element. fn is called with (index, element) and this bound to the element.

$('li').each(function(i, el) { $(el).text(`Item ${i}`); });
.map(fn)

Creates a new collection from the return values of fn(index, element).

.add(selector) / .contents() / .slice(start[,end]) / .end()

.add() merges another selection. .contents() gets child elements. .slice() returns a subset. .end() unwinds a pushed stack.

DOM Manipulation

CORE
.html([value])

Get or set innerHTML. Setter accepts a string or fn(index, currentHTML).

$('#box').html() // get $('#box').html('<b>New content</b>') // set
.text([value])

Get combined text content of all matched elements, or set textContent (XSS-safe). Accepts string or function.

.val([value])

Get or set form element values. Handles checkboxes (true/false), radios, and multi-select (returns array).

.append(...content) / .prepend(...content)

Insert content at the end or beginning of each element. Content can be HTML string, element, or RabbitCollection.

.appendTo(target) / .prependTo(target)

Append/prepend the collection into the target.

.after(...content) / .before(...content)

Insert content after or before each matched element (as siblings, not children).

.insertAfter(target) / .insertBefore(target)

Insert the collection after or before the target element.

.wrap(wrapper) / .wrapAll(wrapper) / .wrapInner(wrapper) / .unwrap([selector])

.wrap() surrounds each element. .wrapAll() wraps all elements together in one wrapper. .wrapInner() wraps each element's children. .unwrap() removes the parent wrapper.

.remove([selector]) / .detach([selector]) / .empty()

.remove() deletes elements from DOM. .detach() removes but keeps the collection reference. .empty() clears innerHTML.

.clone([withEvents])

Deep-clones elements. Pass true to also copy event listeners.

.replaceWith(content) / .replaceAll(target)

Replace matched elements with new content, or replace target with the collection.

Attributes & Data

CORE
.attr(name [, value])

Get or set HTML attributes. Pass an object to set multiple. Pass null as value to remove.

$('img').attr('src') // get $('a').attr('href', '/new') // set $('input').attr({ type: 'text', name: 'q' }) // set multiple $('div').attr('id', null) // remove
.removeAttr(names)

Removes one or more attributes (space-separated names supported).

.prop(name [, value]) / .removeProp(name)

Get or set DOM properties (e.g. checked, disabled, value). Unlike .attr(), this accesses the live property.

.data([key [, value]]) / .removeData([key])

Store arbitrary data against elements using a WeakMap (no DOM pollution). Automatically reads data-* attributes as fallback. Pass no args to get all data.

$('#el').data('user', { id: 1, name: 'Alice' }); $('#el').data('user'); // { id: 1, name: 'Alice' }

Classes

CORE
.addClass(names) / .removeClass([names]) / .toggleClass(names [, state]) / .hasClass(name)

Manage CSS classes. All methods accept a function callback fn(index, currentClass). .toggleClass(name, true/false) forces add or remove. .removeClass() with no arg clears all classes. .hasClass() returns boolean.

$('.card').addClass('active highlight'); $('.card').toggleClass('visible', isLoggedIn); $('.card').hasClass('active'); // true/false

CSS & Styles

CORE
.css(prop [, value])

Get computed style or set inline style. Supports CSS custom properties (--var). Pass an object to set multiple properties. Accepts a function callback.

$('#box').css('color') // get computed $('#box').css('color', 'red') // set $('#box').css({ color: 'red', fontSize: '16px' }) // set multiple $('#box').css('--accent-color', '#ff0') // CSS variable
.cssVar(name [, value])
NOVA

Get or set a CSS custom property (--variable) on the element. Falls back to document.documentElement for reads.

.rootVar(name [, value])
NOVA

Read or write a CSS custom property on :root (document.documentElement), affecting the entire page.

.show([display]) / .hide() / .toggle([show])

Show/hide elements. .hide() remembers the previous display value so .show() restores it correctly. .toggle(bool) forces a state.

Dimensions & Position

CORE
.width([v]) / .height([v]) / .innerWidth() / .innerHeight() / .outerWidth([margin]) / .outerHeight([margin])

Get/set dimensions. width/height use getBoundingClientRect (includes transforms). innerWidth/Height = clientWidth/Height (includes padding). outerWidth/Height = border-box; pass true to include margin.

.offset() / .position()

.offset() returns { top, left } relative to the document (includes scroll). .position() returns { top, left } relative to the offset parent.

.scrollTop([v]) / .scrollLeft([v])

Get or set scroll position. Falls back to window.scrollY/X for document-level queries.

.scrollIntoView([options])
NOVA

Smoothly scrolls the element into view. Default: { behavior: 'smooth', block: 'start' }.

Events

CORE
.on(types [, selector], fn [, options])

Attach one or more event handlers. Multiple event types can be space-separated. Supports namespacing ('click.myns') and event delegation via the optional selector argument.

$('.btn').on('click', handler); $('#list').on('click', 'li', handler); // delegated $('.btn').on('mouseenter.ui mouseleave.ui', handler);
.off([types [, selector], fn])

Remove event handlers. Supports namespaces and selectors. Call with no args to remove all handlers from all elements.

.one(types [, selector], fn)

Attaches a handler that fires at most once per element, then removes itself.

.trigger(type [, data]) / .triggerHandler(type [, data])

.trigger() dispatches a bubbling CustomEvent on all elements. .triggerHandler() calls only the first registered handler on the first element without bubbling.

.hover(inFn [, outFn])

Shorthand for mouseenter/mouseleave. If only one function is given, it handles both.

.onThrottle(event, fn, wait) / .onDebounce(event, fn, wait)
NOVA

Attach throttled or debounced event listeners directly on the collection. Default waits: 200ms throttle, 300ms debounce.

Shorthand event methods (call with no arg to trigger, with fn to bind):

.click(fn) .dblclick(fn) .focus(fn) .blur(fn) .change(fn) .input(fn) .submit(fn) .keydown(fn) .keyup(fn) .keypress(fn) .mouseenter(fn) .mouseleave(fn) .mouseover(fn) .mouseout(fn) .mousemove(fn) .mousedown(fn) .mouseup(fn) .contextmenu(fn) .scroll(fn) .resize(fn) .load(fn) .error(fn) .select(fn)

Animation

CORE
.animate(props, duration, easing, callback)

Animates CSS properties using CSS transitions. duration in ms or named string ('slow', 'fast'). Returns a chainable collection; use .then(fn) to run code after completion. Also accepts an options object as second argument.

$('#box') .animate({ opacity: 0, left: 200 }, 600, 'ease-in-out') .then(() => console.log('done')); // Options object form $('#box').animate({ height: 0 }, { duration: 400, easing: 'ease', complete: () => $('#box').hide() });
.stop([clearQueue, jumpToEnd])

Immediately stops the current CSS transition on matched elements.

.fadeIn() / .fadeOut() / .fadeTo(duration, opacity) / .fadeToggle()

Opacity transitions. All accept optional (duration, easing, callback).

.slideDown() / .slideUp() / .slideToggle()

Height transitions that reveal or hide content vertically. All accept optional (duration, easing, callback).

.timeline(steps)
NOVA

Sequences multiple animations with optional delays. Each step: { target, props, duration, easing, offset } where offset adds extra delay before that step.

$('body').timeline([ { target: '#a', props: { opacity: 1 }, duration: 300 }, { target: '#b', props: { left: '100px' }, duration: 400, offset: 100 }, { target: '#c', props: { opacity: 1, top: '0px' }, duration: 500 }, ]);
$.keyframes(name, frames)
NOVA

Programmatically injects a @keyframes rule into the first stylesheet. Properties are auto-converted from camelCase.

$.keyframes('pop', { '0%': { transform: 'scale(1)' }, '50%': { transform: 'scale(1.3)' }, '100%': { transform: 'scale(1)' }, }); $('.btn').css('animation', 'pop 0.4s ease');

Forms

CORE
.serialize()

Encodes a form as a URL-encoded string (name=value&name2=value2). Works on <form> or any element inside one.

.serializeArray()

Returns an array of { name, value } objects for all form fields.

.serializeJSON()
NOVA

Serializes the form directly to a plain JavaScript object. Multiple values for the same name are collected into an array.

const data = $('#myForm').serializeJSON(); // { username: 'alice', tags: ['js', 'css'] }
.submit([fn]) / .reset()

Bind or trigger form submit. .reset() resets all fields to their default values.

AJAX

CORE
All AJAX methods use the native fetch API and return native Promises. $.ajax() is jQuery-compatible and accepts the same options object.
$.ajax(url [, options])

Performs an async HTTP request. The options object accepts: method, data, headers, contentType, dataType ('json'/'html'/'xml'/'text'), timeout, cache, crossDomain, beforeSend, success, error, complete. Returns a Promise with an .abort() method.

$.ajax({ url: '/api/users', method: 'POST', contentType: 'application/json', data: { name: 'Alice' }, success: res => console.log(res), error: (xhr, status, msg) => console.error(msg), });
$.get(url [,data] [,fn]) / $.post(url [,data] [,fn]) / $.put() / $.patch() / $.delete()

Shorthand methods for specific HTTP verbs. All return a Promise.

$.getJSON(url [, data] [, fn])

Shorthand for a GET request that parses the response as JSON.

$.getScript(url [, fn])

Dynamically loads and executes a JavaScript file.

.load(url [, data] [, fn])

Fetches HTML from url and injects it into matched elements. If data is provided, uses POST. Callback receives (html, status).

$.ajaxSetup(options)

Sets default options for all subsequent $.ajax() calls.

Deferred & Promises

CORE
$.Deferred([fn])

Creates a Deferred object — a chainable utility for async operations. Full jQuery 3 API: .done(), .fail(), .always(), .progress(), .then(), .resolve(), .reject(), .notify(), .resolveWith(), .rejectWith(), .state().

const d = $.Deferred(); d.done(val => console.log('resolved:', val)) .fail(err => console.error(err)); d.resolve('success');
$.when(...deferreds)

Returns a new Deferred that resolves when all passed Deferreds resolve, or rejects if any rejects. Analogous to Promise.all.

$.Callbacks(flags)

A multi-purpose callback list. Flags: 'once' (fire once), 'memory' (replay on new add), 'unique' (no duplicates), 'stopOnFalse'. Methods: .add(), .remove(), .fire(), .fireWith(), .has(), .disable(), .lock().

Reactive Binding

NOVA
Reactive binding lets you attach observable state directly to DOM elements. When a value changes, a render callback fires automatically — like a micro-Vue, without any framework.
.bind(key, renderFn)

Declares a reactive property key on each element. Whenever that property is set, renderFn(value, $el) is called with this bound to the DOM element.

$('#counter').bind('count', function(val, $el) { $el.text(`Count: ${val}`); $el.toggleClass('danger', val > 10); });
.setReactive(key, value)

Updates the reactive property, triggering the render callback.

$('#counter').setReactive('count', 5); // triggers renderFn
.reactive(key)

Reads the current value of a reactive property from the first element.

Observers

NOVA
.observe(fn [, options]) / .inView(fn [, threshold])

Uses IntersectionObserver to call fn(entry, $el) when visibility changes. .inView() is a shorthand with a default threshold of 10%.

$('.section').inView(function(entry, $el) { if (entry.isIntersecting) $el.addClass('visible'); }, 0.25); // 25% visible before triggering
.watch(fn [, options]) / .unwatch()

Attaches a MutationObserver to each element. Calls fn(mutations, $el) when the subtree changes. Default options watch childList, attributes, subtree, and characterData. .unwatch() disconnects.

.onResize(fn)

Attaches a ResizeObserver. fn(entry, $el) is called whenever the element is resized.

.lazyLoad([options])

Lazy-loads images using IntersectionObserver. When the image enters the viewport, the data-src attribute (configurable) is moved to src. Options: { threshold, src, placeholder }.

$('img[data-src]').lazyLoad({ placeholder: '/spinner.gif', threshold: 0.2, });

Drag & Drop

NOVA
.draggable([options])

Makes elements draggable via mouse events. Options: handle (selector for drag handle), start, drag, stop callbacks.

$('#panel').draggable({ handle: '.panel-header', start: e => console.log('drag started'), stop: e => console.log('dropped at', e.clientX, e.clientY), });
.droppable([options])

Makes elements drop targets. Options callbacks: over, drop, enter, leave.

$('#dropzone').droppable({ over: () => $('#dropzone').addClass('over'), drop: e => console.log('dropped!', e.dataTransfer), });

Template Engine

NOVA
$.template(tpl, data)

A lightweight mustache-like template engine. Supports variable interpolation, conditionals, and loops.

SyntaxDescription
{{ key }}Interpolate value (dot notation supported)
{% if key %} ... {% endif %}Conditional block
{% each list as item %} ... {% endeach %}Loop over array
const html = $.template(` <h1>{{ user.name }}</h1> {% if user.admin %} <span>Admin</span> {% endif %} {% each items as item %} <li>{{ item }}</li> {% endeach %} `, { user: { name: 'Alice', admin: true }, items: ['a', 'b'] }); // Or render directly into elements $('#app').render(tplString, data);

Event Bus

NOVA
$.bus

A global pub-sub event bus for cross-component communication, independent of the DOM.

$.bus.on('user:login', user => console.log('logged in', user)); $.bus.once('app:ready', () => $('#loader').hide()); $.bus.emit('user:login', { id: 1, name: 'Alice' }); $.bus.off('user:login', handler); // remove specific handler

HTTP Client

NOVA
A mini Axios-style HTTP client with request/response interceptors and automatic retry logic. Separate from $.ajax.
$.http.get/post/put/patch/delete(url [, data [, opts]])

Makes a fetch request with JSON content type. Returns a native Response Promise.

const res = await $.http.post('/api/login', { email, password }); const data = await res.json();
$.http.fetch(url, opts, retries)

Low-level fetch with retry. Retries with exponential back-off (300ms × attempt).

$.http.interceptors.request.use(fn) / $.http.interceptors.response.use(fn)

Register middleware that runs before every request or after every response. Useful for auth headers or global error handling.

$.http.interceptors.request.use(opts => { opts.headers = { ...opts.headers, Authorization: `Bearer ${token}` }; return opts; });

Storage & Cookies

CORE
$.store.set/get/remove/clear

localStorage wrapper with automatic JSON serialization. Pass true as the last argument to use sessionStorage instead.

$.store.set('prefs', { theme: 'dark' }); $.store.get('prefs'); // { theme: 'dark' } $.store.remove('prefs'); $.store.set('session-key', val, true); // sessionStorage
$.cookie.set/get/remove/all

Cookie helpers. .set(name, value, days, path) creates a cookie with optional expiry. .all() returns all cookies as a plain object.

$.cookie.set('token', 'abc123', 30); // 30 days $.cookie.get('token'); // 'abc123' $.cookie.remove('token');

Utilities

CORE
$.throttle(fn, wait) / $.debounce(fn, wait)

Returns a rate-limited version of fn. Throttle ensures fn fires at most once per wait ms. Debounce delays execution until wait ms after the last call.

.copy()

Copies the text content or value of the first element to the clipboard. Returns a Promise. Uses Clipboard API with fallback.

$.each(obj, fn) / $.map(obj, fn) / $.grep(arr, fn [, invert])

Utilities for iterating arrays and objects. $.grep filters an array; pass true as third arg to invert.

$.extend([deep,] target [, ...sources])

Merges sources into target. Pass true as first arg for deep merge. With one arg, merges into $.fn (plugin registration).

$.isFunction / $.isArray / $.isString / $.isObject / $.isNumeric / $.isPlainObject / $.isEmptyObject / $.isWindow

Type-checking utilities. All return a boolean.

$.type(value)

Returns a lowercase string type: 'array', 'object', 'function', 'null', etc.

$.proxy(fn, context [, ...args])

Returns a function that always runs with this === context. Pre-applied args are prepended.

$.parseJSON / $.parseXML / $.parseHTML

$.parseJSON is an alias for JSON.parse. $.parseXML uses DOMParser. $.parseHTML returns an array of DOM nodes from an HTML string.

$.uuid()

Generates a unique ID string with prefix rb-.

$.camelCase(str) / $.globalEval(code) / $.contains(a, b) / $.inArray(val, arr) / $.merge(a, b) / $.unique(arr)

Miscellaneous utilities mirroring the jQuery static API. $.camelCase converts kebab-case to camelCase. $.globalEval executes a string in global scope.

Legacy Event API

JQUERY
.bind(types, fn) / .unbind(types [, fn])

jQuery 1.x event binding. Aliases for .on() / .off().

.delegate(selector, types, fn) / .undelegate(selector, types [, fn])

jQuery 1.7 delegated event binding. Aliases for .on(types, selector, fn) / .off(types, selector, fn).

.live(types, fn) / .die(types [, fn])

Deprecated jQuery 1.x global delegation via document. Included for legacy plugin compatibility.

$.event

Internal event hook object. Contains special, fix, add, remove, and trigger — used by jQuery UI and other low-level plugins.

Callbacks & Deferred

JQUERY

See Deferred & Promises section — $.Callbacks and $.Deferred are fully implemented there with the complete jQuery 3 API.

Queue System

JQUERY
.queue([type] [, data]) / .dequeue([type]) / .clearQueue([type])

Manage the function queue for elements. Default queue name is 'fx'. Plugins that chain animations using jQuery's queue system will work correctly.

.promise([type])

Returns a Deferred that resolves when all queued functions have finished running on the collection.

Hooks & Animation Settings

JQUERY
$.cssHooks

An object where plugins can register custom CSS property handlers. Each key is a CSS property name with optional get(el) and set(el, value) hooks.

$.cssNumber

A map of CSS properties that should not have px appended when set with a number (e.g. opacity, zIndex).

$.fx

Animation settings object. $.fx.off = true disables all animations globally (useful for testing). $.fx.speeds maps named durations: 'slow' = 600ms, 'fast' = 200ms.

$.fx.off = true; // disable all animations (e.g. in tests) $('#el').animate({ opacity: 0 }, 'slow'); // uses 600ms
$.holdReady(hold) / $.readyException(error)

Delay the DOM ready event until external resources are loaded ($.holdReady(true) / $.holdReady(false)). Used by AMD loaders and script managers.

Writing Plugins

RabbitJS is a full drop-in for jQuery. Any existing jQuery plugin will work. Writing new plugins is identical to the jQuery pattern.
// Standard jQuery plugin pattern — works unchanged (function($) { $.fn.highlight = function(color) { return this.each(function() { $(this).css('background', color || 'yellow'); }); }; })($); // Use it $('.important').highlight('#ffe44e'); // Alternative: $.fn.extend $.fn.extend({ shake() { return this.animate({ left: 10 }, 80) .animate({ left: -10 }, 80) .animate({ left: 0 }, 80); } }); // Or use the RabbitJS static plugin method $.plugin('blink', function() { return this.fadeOut(500).fadeIn(500); });

Plugin compatibility checklist. RabbitJS provides everything well-behaved jQuery plugins need:
